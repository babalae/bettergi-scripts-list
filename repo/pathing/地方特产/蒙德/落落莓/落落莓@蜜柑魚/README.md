# 落落莓采集路线 · 推荐配置与说明

## 推荐使用角色：迪希雅（行走位）

- **建议等级**：迪希雅 60 级（突破后）
  - 解锁固有天赋「崇诚之真」
    - 效果：提升队伍白天移动速度，采集更高效

- **推荐武器**：钟剑
  - 效果：受到攻击时生成护盾
  - 用途：在怪物群中穿梭时提供额外保护

## 路线说明

- 采集路径中 **经常需要在怪物群中穿梭**
- 在落落莓附近有怪物群聚集的点位，路线会自动使用迪希雅的 **元素战技（E）**
  - 提供抗打断效果，防止角色被打偏而无法对准采集点

## 注意事项

- **A02 路径** 包含简易脚本，会尝试走向地图边缘采集
  - 该路径 **采集失败概率较高**，请谨慎使用或手动调整

## 推荐队伍配置

| 角色 | 作用 |
|------|------|
| 迪希雅 | 行走位、抗打断、白天移速辅助 |
| 罗莎莉亚 | 夜间移速提升（与迪希雅形成全天候加速） |
| 枫原万叶 | 体力消耗降低、聚怪辅助 |
| 任意风系角色 | 双风共鸣：提升移速、降低体力消耗 |

> 双风 + 迪希雅（白天）+ 罗莎莉亚（夜间）可实现全时段高速采集体验。
