欢迎你使用由[Annijang](https://github.com/Annijang)（qq是2776101087）制作的地图追踪脚本  
由于本人是第一次制作脚本，有的地方可能还不熟练导致卡墙，但本人在录制完脚本后均让BGI自动运行了一遍脚本，但可能因为每个人不同的战斗策略导致出现卡墙的问题属正常情况，但如果你有多次运行之后均已发现有关地图追踪脚本的卡墙问题请联系我。
本地图追踪脚本使用[BetterG 0.54.0]制作，请勿使用更低版本的BGI或者其他非[BGI官方](https://bettergi.com/)发布的版本的BGI软件运行此脚本，否则因兼容性等导致的一切问题请联系对应版本的作者或者升级BGI版本进行解决。

测试环境：【1920x1080 窗口】、60帧

感谢[空荧酒馆](https://yuanshen.site/docs/)的点位信息为我制作地图追踪脚本提供的帮助。

丘丘人射手：由于丘丘人射手的位置在高处，导致掉落材料很容易捡不到，请务必在战斗配队里携带一个万叶或者琴来聚集材料，如有必要请务必在“全自动”->“调度器”->“配置组”->“{你的配置组}”->右上角“设置”中在“战斗配置”下方打开“自动拾取掉落物”或者“聚集材料动作”选项，这将能尽可能让掉落物被拾取而不是路过就走了。携带万叶能够拾取到80%-90%的材料，不携带万叶只能拾取差不多30%-40%的材料。

注：由本人制作的地图追踪脚本在运行时会提前释放部分角色的元素战技，跟JS脚本【锄地一条龙】的泥头车功能类似，所以请务必在配队里至少携带“芙宁娜、玛薇卡、爱可菲、伊涅芙”中的其中1名角色。
