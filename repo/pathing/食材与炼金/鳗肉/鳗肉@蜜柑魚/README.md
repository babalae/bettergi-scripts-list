## 路径说明

### 路径构成
- **稻妻部分**：主要取自「提瓦特食材一条龙」中的 `C稻妻食材@MissDan`  
  该路径沿途会采集其他材料，**执行时间会比纯鳗肉路径稍长**

- **须弥、渊下宫部分**：主要参考 `鳗肉@海豹穹穹队` 路径修改制作

---

### 注意事项

1. **队伍配置**  
   队伍中必需包含 **瑶瑶 / 早柚 / 蓝砚** 等不会惊动鳗鱼的角色
   若使用 **绮良良** 或 **埃洛伊**，还能避免因附近鸟类受惊而连带吓跑鳗鱼

2. **沙漠环境**  
   若使用「沙漠后缀」路径，建议在**沙漠无沙暴时执行**，否则可能影响路径正常运作  
   *推荐配合「采集CD管理」使用*

3. **特殊机制**  
   「神的棋盘」路径中有两条鳗肉埋在**沙堆**中，需要队伍中有**风系角色**才能采集

4. **操作提醒**  
   执行过程中尽量避免在鳗鱼点附近释放生存位技能，否则**可能导致鳗鱼受惊逃窜**

5. **不可抗力因素**  
   以下情况可能导致鳗鱼受惊逃窜，造成空跑：
   - **怪物镇守的鳗鱼点**（仍可能拾取部分鳗肉）
   - **下雨天气**（雨水会惊扰鳗鱼）
