Tips：
①本路线耗时约；
②本脚本在运行JS脚本【锄地一条龙】后更安全稳定；
③果树部分会使用到万叶，如果不携带万叶也同样不影响其他部分食用；
④无迪希雅的小伙伴可以使用钟剑大剑行走位代替，依靠钟剑挨揍的盾足以应对本脚本大多数怪堆场景；
