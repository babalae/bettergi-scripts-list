## 配置与运行要求

> 虽然全线会路过两只霜夜灵嗣，但是不用战斗，都是直接冲过去的哦。

### 🎯 角色要求
- 确保队伍中有「菈乌玛」与「枫原万叶」
- 将`行走位`设置为「菈乌玛」

### ⚠️ 运行须知（必看）
- 运行前请确保游戏内不在下雨状态，下雨时挪德卡莱的小动物会消失。运行中途若开始下雨，将无可避免地白跑。
- 运行前请确保你今天没有跑过虹滴晶路线和霜夜灵嗣路线，这两个路线与犀牛路线存在重叠，会导致犀牛跑走。

### 🚀 运行效率
- 9分钟可收获理论最多83个兽肉
- 常态下拾取约70个

### ⚔️ 风险点位说明
最后一条路径落地点有一只精英怪，拾取时会触发入战，但是不会打起来！只会拾取！可选择性关闭，这个点位物资丰厚，高风险高回报
