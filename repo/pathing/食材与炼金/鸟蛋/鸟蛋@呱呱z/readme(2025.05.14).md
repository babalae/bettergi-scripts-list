# 鸟蛋自动采集脚本

## 介绍

本脚本旨在为用户提供自动采集鸟蛋的辅助工具。鸣谢：@不瘦五十斤不改名，@灰林鸮。

## 脚本功能

- 自动整合仓库内的鸟蛋采集脚本；
- 按照国家、地区及拖图顺序进行采集任务整理。
