## 四、脚本食用建议
- 食材刷新时间为每日0点，螃蟹类刷新时间为采集后12小时，推荐间隔12小时且经过每日0点后运行本脚本；
- 本脚本在运行JS脚本【锄地一条龙】后更安全稳定；
- 果树部分刚需万叶，不携带万叶不影响其他部分食用;
- 无迪希雅的情况，可使用其他大剑行走位代替，依靠钟剑挨揍的盾足以应对本脚本大多数怪堆场景，但有一定概率被电气水晶小卡脚；
