配置组设置注意事项
1、开启地图追踪行走配置
2、关闭自动拾取
3、开启战斗配置
4、战斗策略选择"根据队伍自动选择"
5、关闭自动检测战斗结束
6、其他默认就行

JS配置注意事项
1、必须同时填写复活料理和恢复料理，才能正常复活
2、目前bgi本体不支持战斗复活，检测到阵亡角色会立刻报错，并传送到七天神像复活，脚本中的复活功能是每轮战斗提前做了一次检测，因此战斗脚本中最好每个角色只上场一次，避免复活检测失效。
3、单轮战斗时长是指战斗脚本运行一轮所需要的时间，填写战斗时长可以精确到小数点后两位，我最常用的周本队大约是13.5秒，不要填个几百秒之类的，不然很多功能都用不了
4、战斗中只有四类料理可以生效，重复使用相同类型的料理会覆盖掉上一个，四类料理分别为攻击类、防御类、药剂类、冒险类
5、需要勾选最后的用户协议才能正常运行

其他重要注意事项
1、周本大部分流程需要护盾角色提前开盾，请将护盾角色放在第一位！！！
2、世界树博士必须自己录制键鼠脚本，详情请看下方说明!!!
3、如果勾选了红血，自动吃药，请确保4个小道具中有食物袋，并且食物袋中有复活药和恢复料理，
复活药推荐2星以上，1星道具复活后因为血条过短，可能导致红血无法识别无法自动吃药

"博士"：三月博士开局需要前往柱子处躲避，在没有移速加成的情况下少女体型到成男可以正常跑到柱子处躲避，如果你无法到达正确的位置，可以使用记事本打开脚本文件夹中的utils.js文件，使用ctrl+f搜索，找到await keyMaintain("d", 4500); 更改这里的数值，数字越大，向右跑的距离越多。

"世界树博士"：世界树版博士的二阶段机制很特殊，boss会弹出墙壁把人送走，没有简单通用的破解流程，单独训练个模型来实现很麻烦。我的思路是不使用大招先通过一阶段，之后进入二阶段后，这时角色位置刷新重置，可以使用键鼠脚本来完成一次机制破除和一轮输出，练度没问题可以一套直接把boss带走，避免二次弹出墙壁。接下来我分享下我多年键鼠脚本的录制经验。
1、首先千万不要移动你的鼠标，只能使用wsad来实现移动
2、按键和点击可以多来几次提高容错，避免卡顿导致按键和点击被吞掉
3、实战录制的时候，如果你已经成功击败了 boss，不要着急停下录制，再多录制一个轮次，还是为了提高容错
接下来是录制的详细思路，首先等待10s左右，避免上一轮技能还在cd，
之后点击任意一个数字（1到4），确定一个固定的跑图人员，最好是带护盾，先开一个护盾获得霸体效果（钟离的话需要先面向左边，避免长e柱子挡路），
然后按下w走向boss，boss激活后，马上按下s往后慢慢走，直到碰到墙壁，然后按下d，当 boss 弹出墙壁推走角色的时候，多次点击鼠标右键冲刺快速抵达角落，可以多冲刺几次，保证容错
确保到达指定位置后，快速进行普攻破除墙壁，之后按下s键，多次点击鼠标右键冲刺到达 boss 附近，之后就开始正常的输出流程，直到带走 boss。
键鼠脚本录制好了之后，将其命名为"世界树博士战斗",文件格式后缀是json,不要把文件格式也改了。
最后最重要的一点，录制完后一定要测试一下！！！

结语：
本脚本适合不想再花时间手打周本，同时队伍练度较高的玩家，由于BGI的战斗目前还很死板以及boss机制问题，高难度要想挑战成功，基本要靠很高的练度以及合适的角色(脱手远程输出、高额护盾)还有吃药，挑战失败是很正常的，成功率有个30%就很不错了，4.0新版本后更新了红血自动吃药的功能(非常感谢 mno 和 汐 在锄地一条龙中提到的血量检测方法！！！)，大部分场景挑战会更加轻松一些，
最后实在打不过的，可以尝试减低难度，boss材料掉落概率从难度4级到1级为，2.4、2.1、1.55、1。
未来会考虑添加更稳定准确的对应策略，提高挑战成功率。目前在计划中的就是”门扉前的弈局“，检测到必杀技后，会快速前往中心位置使用冰冻破解机制。
