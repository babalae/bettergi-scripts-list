传送并自动秘境 (增强版)



这是一个 BetterGI 的自定义脚本，基于官方示例修改并由 AI 辅助生成。它可以帮助你一键传送到指定的秘境门口，并按照你设定的次数和树脂类型自动刷取副本。



主要功能：

本脚本解决了原版脚本无法直接在 JS 调度器中调整“刷取次数”和“树脂类型”的问题。现在你可以直接为每个脚本任务独立设置这些参数，而无需修改 BetterGI 的全局设置。



如何使用



安装脚本



在 BetterGI 目录下的 User/JsScript 文件夹中新建一个文件夹。



将 manifest.json, main.js, settings.json 三个文件放入该文件夹。



在 BetterGI 的“调度器”或“脚本”界面点击刷新。



配置参数详细说明

点击脚本旁边的“齿轮”图标即可看到以下设定项目：



指定要前往的秘境 (自动传送到刷取的秘境)



从清单中选择你要刷的副本。



自动切换到指定队伍



输入你在游戏内手动设置的队伍名称。



注意：名称必须与游戏内左上角显示的完全一致。留空则不会切换队伍。



刷取至树脂耗尽



勾选 (开启)：执行“刷取至树脂耗尽”策略。优先使用 原粹树脂，用完后使用 浓缩树脂。此模式下不会使用须臾与脆弱树脂，下方的指定次数将被忽略。



不勾选 (关闭)：脚本会按照下方设定的“原粹”、“浓缩”、“须臾”、“脆弱”树脂次数进行刷取，刷取完所有指定次数的树脂才会停止。



各类树脂刷取次数 (原粹/浓缩/须臾/脆弱)



仅在“刷取至树脂耗尽”不勾选时生效。设定对应树脂要刷几场。



结束后自动分解圣遗物



开启后，会在所有战斗结束后自动执行圣遗物分解。



需要快速分解圣遗物的最高星级



选择自动分解时要处理到的最高星级。



战斗完成后等待时间 (秒)



战斗结束后，在寻找石化古树前的延迟时间。建议设为 5 秒以上，等待战斗结算动画与特效完全结束，避免误判。



寻找古树时使用小步伐行走 (正常用户请不要启用)



如果你的电脑性能较差，导致寻找古树时间过久或容易走过头，可以尝试开启此功能。



步行前往开启秘境和领取奖励



如果电脑性能较差，导致开启秘境或者领取奖励的 F 键点击不到，可以尝试此功能，角色会用走的靠近互动点。



寻找古树时确认位置左右移动的次数 (正常用户不要修改)



配合“小步伐行走”使用，设定左右移动确认位置的频率。默认值为 3。



自动吃药



开启后，当角色血量过低（红血）时，脚本会自动按 Z 键。请务必先在游戏中装备“便携营养袋”。



角色死亡后重试次数



在秘境战斗中，若发生角色死亡，脚本尝试复活并重新挑战的次数。



秘境刷取轮数



设定总共要刷几轮。填 0 代表刷到没体力为止（或达到指定树脂次数）。填入数字则刷满指定场数后停止。



周日副本序号



仅在周日（所有天赋/武器素材副本全开）时使用。填入 1、2 或 3 对应界面上的选项顺序。



常见问题



Q: 进本后选错了难度等级（比如选了 Level 1）怎么办？



A: 脚本默认会点击游戏记忆的那个位置。请手动打一次该秘境并选择最高难度（Level 4），游戏记住后，下次脚本就会点对位置了。



Q: 脚本显示“未找到秘境”？



A: 请确认你选择的秘境名称是否在当前版本的 BetterGI 支持列表中。
