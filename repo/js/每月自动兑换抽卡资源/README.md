# 每月自动兑换抽卡资源

## 脚本用途

每个月自动兑换蓝球和粉球，兑换资源不够会提醒(需要打开 js 通知)，本月兑换过会自动跳过，想要重置 CD可以把monthly.txt中的时间删掉

## 资源用途

本目录包含脚本运行所需的资源文件，以下用途根据资源文件名、目录名和 `main.js` 中的资源引用进行保守推断：

- 界面、图标、物品或状态识别模板图片。
  - `assets/blueBall.png`
  - `assets/pinkBall.png`
- 界面按钮或状态识别模板图片。
  - `assets/paimon_menu.png`
- 脚本运行所需的文本配置或状态记录。
  - `assets/monthly.txt`

## 注意事项

- 请保持资源文件的相对路径不变，避免脚本无法加载。
- 若替换识别图片或路径 JSON，请同步检查 `main.js` 中的引用。
- 未能从文件名或引用明确判断的资源，按“用于脚本运行所需的识别或配置资源”处理。
