# 七圣召唤角色邀请全自动

## 脚本用途

挑战前请选好卡组并设好js脚本参数，最少持有一张邀请函或1000牌币！！！建议先手动买，避免一些Bug

## 资源用途

本目录包含脚本运行所需的资源文件，以下用途根据资源文件名、目录名和 `main.js` 中的资源引用进行保守推断：

- 用于脚本运行所需的识别或配置资源。
  - `assets/1.json`
- 界面、图标、物品或状态识别模板图片。
  - `assets/talkSymbol.png`
  - `assets/tavern.png`

## 注意事项

- 请保持资源文件的相对路径不变，避免脚本无法加载。
- 若替换识别图片或路径 JSON，请同步检查 `main.js` 中的引用。
- 未能从文件名或引用明确判断的资源，按“用于脚本运行所需的识别或配置资源”处理。
