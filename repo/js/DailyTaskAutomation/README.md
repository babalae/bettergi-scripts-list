# 每日任务自动化

自动化完成每日委托、合成树脂、领取奖励等日常任务。

## 功能

* **设置世界权限**：自动将世界权限设置为指定状态
* **合成浓缩树脂**：自动前往合成台合成浓缩树脂
* **领取每日委托/探索派遣/历练点**：自动前往冒险家协会领取
* **领取纪行奖励**：自动领取纪行奖励
* **领取邮件**：自动领取所有邮件奖励

## 配置说明

|配置项|类型|说明|默认值|
|-|-|-|-|
|全局重试次数|输入|每个任务失败后的重试次数|3|
|全局超时时间|输入|单个任务超时时间（秒）|60|
|世界权限设置|选择|不允许加入/确认后可加入/直接加入|不允许加入|
|合成浓缩树脂|开关|是否执行合成树脂任务|开启|
|合成树脂地区|选择|蒙德/璃月/稻妻/须弥/枫丹/纳塔|枫丹|
|冒险家协会地区|选择|蒙德/璃月/稻妻/须弥/枫丹/纳塔/挪德卡莱|枫丹|
|领取纪行奖励|开关|是否领取纪行奖励|开启|
|领取邮件|开关|是否领取邮件|开启|

## 使用要求

* 游戏分辨率：1920x1080
* 队伍需为 4 人队伍（部分任务需要）
