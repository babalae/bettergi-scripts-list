# GOOD Scanner

扫描角色、武器、圣遗物，导出 GOOD v3 格式 JSON，可导入 [ggartifact.com](https://ggartifact.com)。

## 使用前提

- BetterGI v0.44.3+
- 游戏语言：简体中文
- 游戏分辨率：16:9（如 1920×1080、2560×1440、3840×2160）
- 当前队伍中**不要**包含达达利亚和丝柯克
- 武器背包排序设置为**品质顺序 - 降序**
- 圣遗物背包**不要使用筛选**，且**不要按获取时间排序**

## 使用方法

1. 在JS脚本自定义设置中填写你的windows文本缩放值(例如100%就填1，125%就填1.25，150%就填1.5)，填入你游戏中主角、流浪者、奇偶的名字
2. 在 BetterGI 中运行本脚本，如果电脑比较慢可以增加每种操作的等待时间
3. 找到保存在此脚本目录下 `records/good_export_<时间戳>.json`的文件
4. 将 JSON 文件导入 ggartifact.com
