## 一、功能概述

### 1. 初始化记录

- 用于建立药品库存的“基准值”
- 通常在第一次使用或库存发生大幅变化时执行
- 会将当前库存写入【history】

### 2. 记录 + 消耗计算

- 在存在初始化记录的前提下运行
- 自动查找最近一次初始化记录
- 计算当前库存相对于初始化时的消耗量
- 记录结果写入【history】
- 消耗结果写入【consumption】

### 3. 纯记录模式

- 不参与初始化与消耗计算
- 每次运行仅记录当前库存
- 用于长期留档或手动分析
- 输出至【snapshot】

## 二、输出文件说明

- 三个模式分别输出三个文件，格式为：年月日时分秒-模式-名称-数量，回血和复活药分行记录。

## 三、使用说明

- 纯记录模式：运行一次记录一次，自行操作记录。
- 消耗对比模式：普通记录减去上次初始化记录，得出消耗量，如果没有初始化，则自动转化为初始化数据。

## 四、版本与环境
