# 自动领取每日奖励与自定义通知

一个基于 BetterGI / BGI 的《原神》自动化脚本插件。

支持自动完成：

* 探索派遣领取与重新派遣
* 每日委托奖励领取
* 邮件领取
* 珍珠纪行奖励领取
* 树脂与历练点检测
* 每日状态通知

插件会自动识别当前玩家昵称与 UID，并在通知中附带对应信息。

---

## 功能特性

### 自动领取内容

* 冒险家协会探索派遣
* 每日委托奖励
* 邮件奖励
* 珍珠纪行任务奖励
* 珍珠纪行等级奖励

### 状态检测

可自动检测：

* 当前原粹树脂
* 剩余历练点
* 每日委托是否完成

当长效历练点不足时会发送提醒通知。 

### 自定义通知

支持独立开关：

* 邮件领取通知
* 纪行领取通知
* 每日委托通知
* 错误通知
* 每日状态通知

---

## 工作流程

插件启动后会自动执行：

1. 获取当前账号昵称与 UID
2. 前往冒险家协会
3. 领取并重新派遣探索
4. 领取每日委托奖励
5. 领取邮件
6. 领取纪行奖励
7. 检查每日状态并发送通知
