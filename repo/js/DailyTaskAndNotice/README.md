# 自动领取每日奖励与自定义通知

一个基于 BetterGI / BGI 的《原神》自动化脚本。

支持自动完成：

* 探索派遣领取与重新派遣
* 使用历练点完成每日委托奖励领取
* 邮件领取
* 珍珠纪行奖励领取
* 树脂与历练点检测与每日状态通知

脚本会自动识别当前玩家昵称与 UID，并在通知中附带对应信息，建议BGI设置中开启通知附截图功能。

---

## 功能特性

### 状态检测

可自动检测：

* 当前原粹树脂
* 剩余历练点
* 每日委托是否完成

当长效历练点不足时会发送提醒通知。 

### 独立控制

支持功能独立开关与独立通知开关

---

## 工作流程

脚本启动后默认会自动执行：

1. 获取当前账号昵称与 UID
2. 前往冒险家协会
3. 领取并重新派遣探索
4. 领取每日委托奖励
5. 领取邮件如果有新邮件则发送领取通知
6. 领取纪行奖励
7. 检查每日状态并发送通知
