# 千星奇域-绮衣珍赏 脚本功能说明

主要功能：全自动完成`千星奇域-绮衣珍赏`并领取奖励，脚本结束后自动返回提瓦特

基于 **OCR · 模板匹配** 

## 使用的千星奇域关卡

本JS采用当前人气奇域《弓箭传说：元素肉鸽》(GUID: 118782821495)

该关卡经过长期测试，在提升奇域内角色属性后具有稳定刷取次数和时长的优点

## 关于奇域内难度选择和奇域竞技段位天梯影响

**重要：**如果追求胜利结算，建议在奇域内商店优先购买攻击力，再购买生命值，其他物品酌情购买（攻击力上限：5000生命上限：200500）

奇域内难度和竞技段位天梯分数关系： 简单(+60) 困难(+90) 地域(+150) 无尽深渊(+150-200)  
如果仅用于刷取`绮衣珍赏`，对天梯分数影响不大，但是仍建议选择简单难度以减少对天梯排名的影响
