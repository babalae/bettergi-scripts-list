# 兽肉好感：自动好感度&刷两只鳄鱼兽肉&卡时间

## 脚本用途

通过突发事件:张牙舞爪的恶党刷兽肉，并顺带获取好感度（好感度超过10次后不再获取），请使用满员好感度队伍，并为全体队伍角色配置相应的战斗策略

## 资源用途

本目录包含脚本运行所需的资源文件，以下用途根据资源文件名、目录名和 `main.js` 中的资源引用进行保守推断：

- 地图追踪或自动路径 JSON，供脚本加载对应路线。
  - `assets/AutoPath/万叶版前往.json`
  - `assets/AutoPath/万叶版返回.json`
  - `assets/AutoPath/好感-张牙舞爪的恶党-循环(二净甸).json`
  - `assets/AutoPath/好感-张牙舞爪的恶党-循环(二净甸刷肉版).json`
  - `assets/AutoPath/好感-张牙舞爪的恶党-触发位置(二净甸).json`
- 用于脚本运行所需的识别或配置资源。
  - `assets/run_record.json`
  - `assets/万叶拾取.json`

## 注意事项

- 请保持资源文件的相对路径不变，避免脚本无法加载。
- 若替换识别图片或路径 JSON，请同步检查 `main.js` 中的引用。
- 未能从文件名或引用明确判断的资源，按“用于脚本运行所需的识别或配置资源”处理。
