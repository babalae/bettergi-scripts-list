## 1.下拉列表模式

### 1.1.使用前准备

确保你想要切换的账号是已登录状态，且账号选择列表中存在。

### 1.2.使用方式

1. 把此脚本添加到配置组
2. 右键点击选择"修改JS脚本自定义配置"
3. 填入你想登录的账户名称，该名称必须和账号列表中显示的一致
   - 请确认你此次登录使用的是手机号还是邮箱

## 2.B服切换另一个账号

确保你想要切换的账号（目前只支持两个账号互相切换）是已登录状态。
