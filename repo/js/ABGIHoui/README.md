## 使用方法

#### 在`调度器`中新建`配置组`，`配置组`添加`ABGIHoui`右键脚本名字，找到【修改通用配置】将里面的【JS HTTP权限】下的`禁用`选择`启用`，在【修改JS脚本自定义设置】，选or填相关内容，最后一张图是修改key以及网页登录账号和密码
