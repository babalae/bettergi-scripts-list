# 防休眠倒计时

在游戏主界面**等待 / 倒计时**，并在等待期间**防止电脑自动休眠、熄屏**。到点后脚本自动结束，由调度器接着执行下一个任务。

适合的场景：晚上挂着，等到**版本更新 / 整点 / 凌晨重置**之后再自动开始日常，又不希望电脑在漫长等待中睡过去。

> 与仓库已有的「等待到指定时间」脚本的区别：本脚本在等待期间会**周期性轻晃镜头保持系统唤醒**，并且支持「相对倒计时」模式与可被「停止」及时打断。

---

## 一、功能说明

- **两种等待模式**（二选一）：
  1. **等到指定时钟时间**（优先）：填 `HH:mm`（如 `04:00`），等到当天该时刻。若该时刻**今天已过/已到**：默认**立即结束并继续后续任务**（避免脚本启动稍晚就误等约 24 小时）；如需等到次日，请把「目标已过时是否等到明天」设为「是」。
  2. **相对倒计时**：填分钟 + 秒（上面留空时生效）。
- **零漂移**：始终以「目标时间戳 − 当前时间」判断，跨小时、跨天都精准，不累积误差。
- **防休眠**：每隔设定秒数（默认 60）在主界面轻晃一次镜头——只动视角、不动角色、不开菜单，用于刷新系统空闲计时器，防止自动休眠 / 熄屏。
- **可随时停止**：每秒一跳，按下「停止」可及时中断。
- 结束时打印日志，并可选发送一条通知。
- **可选领取月卡**：结束后可自动领取「空月祝福（月卡）」奖励——每日重置后购买月卡的账号会弹出领取窗口，若不处理会卡住下一个任务。调用 BGI 官方 `genshin.blessingOfTheWelkinMoon()`，未购买月卡时安全跳过。

---

## 二、使用方法

1. 把脚本加入**调度器配置组**，并排在你要执行的下一个任务**之前**。
2. 右键脚本 → 修改自定义配置：
   - 想"等到某个时间"：在**「等到指定时钟时间」**填 `04:00` 这类值即可。
   - 想"等一段时长"：上面留空，填下面的**分钟 / 秒**。
3. 运行配置组。脚本会原地等待并防止休眠，到点后自动结束，调度器随即执行下一个任务。

### 前置条件

- 运行前请**停在游戏主界面（大世界）**，角色原地不动。
- 游戏窗口需处于激活状态，防休眠的晃镜头才有效。

---

## 三、注意事项

- **时间基准为本机系统时间**。国服每日 4:00 重置，请确保电脑为北京时间；其它服按电脑时间换算。
- **防的是「系统睡眠 / 熄屏」，不是 Windows 锁屏**。若电源策略很激进（笔记本合盖、低电量强制睡眠、设置了自动锁定），模拟输入可能拦不住——这种情况请到 **电源选项** 把「睡眠」设为「从不」以彻底保险。
- 防休眠间隔应**小于系统休眠时间**（例如系统 10 分钟休眠，间隔就要 < 600 秒）。默认 60 秒可兜底绝大多数设置。
- 本脚本负责「卡到点」「保持唤醒」与「可选领取月卡」，**不处理登录 / 进游戏 / 过公告**。请确保等待期间游戏始终在主界面。
- 领取月卡为可选项（默认关闭）：仅在已购买月卡、且等待终点为每日重置之后时才需开启。它在倒计时结束后执行，完成并返回主界面后才结束脚本。
