# 🎣 提瓦特自动钓鱼（全流程 + 自选）

一款基于 BGI 的全自动钓鱼脚本，支持全提瓦特所有可追踪垂钓点。  
提供灵活筛选、路径控制、自动化钓鱼与冷却记录等完整功能，适用于日常收集、兑换材料获取与全图钓鱼需求。

---

## ✨ 功能概述

脚本可自动追踪并完成 BGI 支持的所有垂钓点，包括路径导航、时间调整、钓鱼流程与冷却记录等完整逻辑。

---

## ⭐ 核心功能

- 🎯 **自由筛选**：按地区、鱼类、鱼饵自定义任务
- 🎁 **材料直钓**：支持直接钓取兑换材料（如「渔获」所需鱼类）
- 🌗 **自动调节时间**：根据目标鱼类自动切换昼夜
- 🧭 **特殊点位支持**：可处理部分无法通过地图追踪到达的钓鱼点
- 🔁 **从指定点继续任务**（需启用全部垂钓点）
- ⚔️ **战斗路线可选**（默认关闭）
- ⏱ **钓鱼超时自定义**
- ⏳ **定时自动停止（最多 24 小时）**
- 💎 **4 点自动领取月卡**（可选）
- 📒 **垂钓点冷却记录**（多人模式记录房主 CD）
- 🧲 **自动拾取**（可选）

---

## ⚠️ 使用注意事项

1. 部分点位包含战斗，队伍中请避免“双风”或“跑步加速（非冲刺）”角色
2. 跑图角色必须为**少男体型**
3. 鱼类 + 鱼饵的选择逻辑：
    - 所有选中的鱼类会加入任务列表
    - 若某鱼饵对应的鱼类未被选中，则该鱼饵对应的所有鱼类会被自动加入
4. 混合路线（路径追踪 + GCM）可能卡死，可在 JS 配置中关闭键鼠路线
5. 请确保 **F2 未被占用**（用于多人模式检测）
6. 鹤观点位支持自动使用「奇特的羽毛」

# 📘 配置说明

## 📂 基础配置

<details>
<summary><strong>筛选垂钓点</strong></summary>

### 钓取特定物品的兑换材料
字段：`path_sort_material`
- 选择后将忽略其他筛选
- 用于快速钓取如「渔获」「赤穗酒枡」「灰河渡手」等材料

### 🗺 地区筛选
字段：`path_sort_area`
- 可多选
- 包含：蒙德、璃月、稻妻、须弥、枫丹、纳塔、至冬、挪德卡莱、层岩巨渊·地下矿区、渊下宫、远古圣山

### 鱼类筛选
字段：`path_sort_fish`
- 可多选
- 若与鱼饵冲突，会自动补全任务列表

### 鱼饵筛选
字段：`path_sort_bait`
- 可多选
- 若鱼饵对应鱼类未被选中，会自动加入

### 禁用键鼠路线
字段：`block_gcm`
- 默认关闭
- 开启后禁用所有 GCM 路线

### 启用战斗路线
字段：`block_fight`
- 默认关闭

### 启用鹤观路线
字段：`block_tsurumi`
- 需拥有「奇特的羽毛」

### 奇特的羽毛使用方式
字段：`tsurumi_method`
- 方式 1：打开背包替换（默认）
- 方式 2：使用快捷更换
</details>

## ⚙️ 进阶配置

<details>
<summary><strong>垂钓点进阶功能 & BGI 内置钓鱼参数</strong></summary>

### 从指定垂钓点继续
字段：`path_continue`
- 仅在启用全部垂钓点时可用

### 自动抛竿未上钩超时
字段：`time_out_throw`
- 默认：10 秒

### 整个任务超时时间
字段：`time_out_whole`
- 默认：300 秒

</details>

---

## 🧰 高级配置

<details>
<summary><strong>满足特殊需求</strong></summary>

### 自动拾取
字段：`启用自动拾取`

### 自动领取月卡（4 点）
字段：`启用4点领取月卡`

### 垂钓点 CD 统计
字段：`启用垂钓点CD统计`
- 默认开启
- 多人模式非房主自动禁用

### 自定义 CD 记录标识
字段：`自定义CD记录标识`
- 留空则默认读取 UID
- 输入任意文本后将使用输入的文本作为自己世界的CD记录标识

### 强制停止运行时间
字段：`小时` + `分钟`
- 两者同时选择才生效

</details>

## 🧪 开发者配置

### 调试时间
字段：`path_time`
- 全天 / 白天 / 夜晚

字段：`developer_log`
- 目前不可用

</details>
