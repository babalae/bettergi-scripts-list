# 说明

该脚本会打每个世界的一只地方传奇-利亚姆和16只飞萤。



- 关于利亚姆：
  1. 没有万叶拾取很难，所以脚本会自己换上，请确保账号上有万叶，否则不要选择拾取！
  2. 操作角：色默认是 提纳里 钟离,一定一定要有钟离，成功率有74%以上！
  当然如果你没有脚本会自动给你换诺艾尔，不过成功率感人，基本是白跑。
  弓箭手不是水就可以，没有小提是脚本会自动给你换安柏，影响的原因是脚本的计算精度不够从而导致失败（作者的水平有限( •̀ ω •́ )）。
  3. 没有解密的水池还没有制作判断的逻辑，抱歉！

- 关于飞萤：
  1. 没有芙芙很难打的全（没有测试伊涅芙），请确保账号上有芙芙而且放到首位！
  2. 操作角色默认是 芙宁娜 雷电将军,  当然如果你没有脚本会自动给你换芭芭拉和凯亚，不过基本也是白跑。
  另一个可以协同就可以，过程在只会放2个人的小技能，没有普通攻击等等！

- 关于路线：
  1. 需要自己先把用的2个角色放到前2个位置，这里暂时不提供这方面的功能。
  2. 具体操作：把需要跑的路线的文件夹放到pathing里面。文件夹的名称放到自定义配置中，不要错字哦。
  目录结构如下：诛贼-地方传奇-利亚姆\pathing\测试\01-甜甜花-蒙德-苍白本-20个.json

- 多人脚本将在启动后持续，直到以下两个条件中的**任一**条件满足
  1. 次数达到预定数值
  2. 持续时间达到预定数值

  如果两个条件均未填写，脚本将在打完133次后停止。注意如果运行这个脚本，过程可能持续20个小时。

  如果只打利亚姆，9个小时左右40W摩拉和5.5W狗粮

  如果只打飞萤，16个小时左右25W摩拉

  一起打可以减少转场的时间，但是也需要21小时！！！

  因为缓慢的进账和面对强敌的复杂性，所有没有相关统计。（应该是暂时的）
